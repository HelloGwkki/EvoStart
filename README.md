<summary>🔽 编辑项目</summary>

1. 拉取仓库
    
    ```sh
    $ git clone https://github.com/HelloGwkki/EvoStart.git
    ```

2. 安装依赖
    
    (npm)
    ```sh
    $ npm i
    ```
    
    或者
    
    (yarn)
    
    ```sh
    $ yarn install
    ```

3. 用你的编辑器打开整个项目
