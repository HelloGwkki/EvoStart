### 安装

1. 拉取仓库
    
    ```sh
    $ git clone https://github.com/evostart/startpage.git
    ```

2. 安装依赖
    
    (npm)
    ```sh
    $ npm i
    ```
    
    或者
    
    (yarn)
    
    ```sh
    $ yarn install
    ```

3. 用你的编辑器打开整个项目


### 生成可部署文件

(npm)
```sh
$ npm run build
```

(yarn)
```sh
$ yarn build
```
